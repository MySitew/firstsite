Convertisseuse de base

Cet outil très simple vous permet d'écrire un même nombre dans différentes bases de numération.

Par exemple, le nombre π s'écrit  :

  • 3.14159... en décimale (base 10),
  • 3.243F6... en hexadécimale (base 16)
  • ou 11.001001... en binaire (base 2).

  • Tapez pi pour π=3.14159..., e pour e=2.71828..., sqrt(2) pour 2 = 1.41421...
  • Utilisez impérativement les lettres majuscules pour les représentations en base > 10.
  • Les opérations algébriques + - * / ^ sont admises, ainsi que des paires de parenthèses. Les fonctions transcendantes acceptées sont sin(), cos(), exp(), log(), atan().
